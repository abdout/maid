# Onboarding Flows

## Office Registration
1. Go to https://maid-app.databayt.org
2. Click "Office Register" card
3. Login: `company@tadbeer.com` / `1234`
4. Fill 3 steps: Basic Info → Location → Review
5. Submit → Redirects to office dashboard

## Maid Registration
1. Login as office admin: `office@tadbeer.com` / `1234`
2. Go to Maids tab → Click "+" button
3. Fill 7 steps: Basic → Job → Experience → Languages → Photo → Bio → Review
4. Toggle "Publish" → Submit

## Super Admin
1. Go to https://maid-web.databayt.org/login
2. Login: `admin@tadbeer` / `1234`
3. **Offices**: `/offices` → View/Approve/Create offices
4. **Maids**: `/maids` → View/Create/Edit/Import maids
